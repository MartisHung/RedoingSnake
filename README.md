# RedoingSnake
A snake game made by C/C++
這是一個用 C/C++ 的貪食蛇遊戲。

## ProJect Status
Not finish

## 安裝

1. clone 存儲庫：
    ```sh
    git clone https://github.com/yourusername/RedoingSnake.git
    ```
2. 進入專案目錄：
    ```sh
    cd RedoingSnake
    ```
3. 編譯程式：
    ```sh
    g++ -o snake main.cpp
    ```
    ```sh
    clang++ snake main.cpp
    ```

## 使用方法

1. 執行遊戲：
    ```sh
    ./snake
    ```
2. 使用方向鍵控制蛇的移動。
